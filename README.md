# 🚚 Urban Delivery Simulation & Optimization System

## 📌 Overview
This project simulates an urban food delivery system, focusing on intelligent assignment of orders to couriers and efficient route planning under time constraints.

The system models a dynamic environment of couriers, orders, restaurants, and a road network, and serves as a foundation for developing and evaluating advanced optimization algorithms for delivery logistics.

---

## 🎯 Project Goals
- Simulate a real-time urban delivery environment
- Model couriers, orders, and road networks as domain entities
- Support dynamic order assignment and route planning
- Provide a clean architectural base for optimization algorithms
- Demonstrate algorithmic thinking in logistics and resource allocation problems

---

## 🧠 Core Concepts
- **Dynamic system** – orders arrive over time and affect existing routes
- **Time constraints** – preparation times, delivery deadlines, and waiting strategies
- **Order bundling** – combining multiple deliveries per courier when possible
- **Optimization under constraints** – balancing efficiency and service quality
